# To-Do List App

A simple web application to manage daily tasks. It allows you to add, edit, delete and mark tasks as completed, with data persistence in the browser.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Instructions to Run the Project

1. Clone the repository
   git clone https://github.com/Torres-Alan/to-do-list-app.git

2. Navigate to the project folder
   cd to-do-list-app

3. Open the index.html file in your browser
   - You can double click the file
   - Or use an extension like Live Server in VS Code

## Features

- Add new tasks
- Edit existing tasks with confirmation by button or Enter key
- Delete tasks
- Mark tasks as completed
- Data is automatically saved in localStorage
